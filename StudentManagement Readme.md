 Student Management System

## Description
This is a simple Java console application.
It manages student details using the Java Collections Framework.

## Features
- Add student details
- Store students using ArrayList
- Search students using HashMap
- Remove duplicate records using Set
- Sort students using Comparator
- Display student details

## Concepts Used
- Java Collections Framework
- ArrayList
- HashMap
- HashSet
- Comparator
- Enhanced for loop
- OOP concepts

## Files Included
- StudentManagement.java
- output.png

## How to Run
1. Compile the program:
   javac StudentManagement.java

2. Run the program:
   java StudentManagement

## Output
The program displays student details in sorted order
and allows fast searching using student ID.
